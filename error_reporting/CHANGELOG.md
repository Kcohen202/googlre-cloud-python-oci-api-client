## 0.29.0

### Breaking changes

- The underlying autogenerated client library was re-generated to pick up new 
  features and resolve bugs, this may change the exceptions raised from various
  methods. (#4695)

- Upgrading to `google-cloud-logging >= 1.4.0` (#4296)
